# Extension Icons

You need to create the following icon sizes:

- icon-16.png (16x16px)
- icon-32.png (32x32px)
- icon-48.png (48x48px)
- icon-128.png (128x128px)

## Design Guidelines

**Style**: Simple, modern icon representing video frame extraction

**Suggested concepts**:
- Film frame with crosshairs
- Video camera with grid overlay
- Play button with frame border
- Layered frames icon

**Colors**: Match the Reframe brand
- Primary: Blue (#3B82F6)
- Dark background: #18181B
- Accent: White

**Tools**: Use Figma, Sketch, or online icon generators like:
- https://icon.kitchen/
- https://www.favicon-generator.org/

## Temporary Workaround

For development/testing, you can use any placeholder images at these sizes. The extension will work without custom icons (Chrome will use default icons).
